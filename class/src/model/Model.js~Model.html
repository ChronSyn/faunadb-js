    Model

    Base class for all models.

    Models represent database instances. They link a FaunaDB class to a JavaScript class.

    The basic format is:

    class MyModel extends Model {
  ... your methods ...
}
// define class name and fields
MyModel.setup('my_models', {
  x: {},
  y: {converter: new RefConverter(MyModel)}
})

    <span><a href="class/src/model/Field.js~Field.html">Field</a></span>s will be constructed and properties will be generated for each field passed to <span><a href="class/src/model/Model.js~Model.html#static-method-setup">setup</a></span>.

    <span><a href="class/src/model/Builtin.js~Class.html#static-method-createForModel">Class.createForModel</a></span> must be called before you can save model instances.
